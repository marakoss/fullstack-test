## Requirements

This project uses:
- Node > v12

<a name="Install"></a>

## Install

    $ git clone git@github.com:marakoss/fullstack-test.git .
    $ npm install


<a name="Usage"></a>

## Usage

To run tests
    
    $ npm test

To spawn HTTP server with basic HTML form, run:

    $ npm start

Then navigate to http://localost:8080/
